# Saeed Ghanem Portfolio

Welcome to my personal portfolio! This project showcases my skills, professional background, and the projects I have worked on.

## About This Project

This portfolio is based on the [Start Bootstrap](https://startbootstrap.com/) source code. I have customized the original template—editing the content and adjusting some design elements to better suit my personal brand and style. 

While the foundation is from Start Bootstrap, I am continuously working to make it uniquely mine.

## Upcoming Updates

I have several exciting updates planned for this portfolio in the near future:
- **Light & Dark Mode:** Implementing a toggle to allow users to switch between light and dark themes for a better viewing experience.
- **Arabic Language Support:** I will be translating and building the Arabic version of this portfolio entirely from scratch. The code and design for the Arabic version will be **100% my own work**, transforming this into a fully personalized and bilingual project.

## Local Development

If you'd like to run this project locally:
1. Clone the repository.
2. Open `index.html` in your preferred web browser.
